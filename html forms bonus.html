HTML FORMS

1. Go here for directions on making your forms live :)

2. Make a file named register.html.

Create a registration form with the following inputs:

  • username:required, make sure your cursor is in this box on page load automatically
  • email:required
  • password:required
  • confirm_password:required
  • a checkbox for "sign me up to the newsletter" that loads checked

The text on the submit button should read "Create New Account!"

3. Make a file named login.html

Add the following inputs:

  • username, make sure it required, ensure the cursor is inside this input on pageloaded
  • email, make it required
  • password, make it required
  • a checkbox that says "Keep me logged in for 2 weeks" and load the page with the checkbox checked
  • a button that submits the form with the text of "Login"

4. Make a file named property-search.html. This will be a form for a property search.

Add all the appropriate inputs you would need to conduct a property search:

  • Property type
  • square footage
  • min_price
  • max_price
  • zipcode
  • number of bedrooms
  • number of bathrooms
  • etc...

The submit button's text should say "Search now"

5. Make a file named checkout.html. This will be a form for a credit card checkout

  • Make all fields required except for the checkbox
  • Add an input for credit card number, expiration month/year, and security code
  • Add inputs for the billing address
  • dd a checkbox that says "Keep this credit card on file for future transactions"
  • The submit button's text should say "Buy Now!"

6. Commit all your work and push to GitHub :)
